[[req_core_fc-limit-definition]]
[width="90%",cols="2,6a"]
|===
^|*Requirement {counter:req-id}* |*/req/core/fc-limit-definition*
^|A |The operation SHALL support a parameter `limit` based upon the following OpenAPI 3.1 fragment:

[source,YAML]
----
name: limit
in: query
required: false
schema:
  type: integer
  minimum: 1
  maximum: 10000
  default: 100
style: form
explode: false
----
|===
